\doxysubsection{Detailed Description}
\hypertarget{dir_4b2c7d6f7844b87512b4f74b6a153f60_autotoc_md25}{}\doxysubsection{\texorpdfstring{ESP-\/\+DSP ESP32-\/\+S3-\/\+BOX-\/\+Lite Board demo application}{ESP-\/\+DSP ESP32-\/\+S3-\/\+BOX-\/\+Lite Board demo application}}\label{dir_4b2c7d6f7844b87512b4f74b6a153f60_autotoc_md25}
The demo applications are developed for the ESP32-\/\+S3-\/\+BOX-\/\+Lite development board and demonstrate the usage of FFT functions from the ESP-\/\+DSP library. This example showcases how to use FFT functionality to process audio stream data. The application record sound from two microphones and show the spectrum from them at display as 2D plot.\hypertarget{dir_4b2c7d6f7844b87512b4f74b6a153f60_autotoc_md26}{}\doxysubsection{\texorpdfstring{Audio Processing Flow}{Audio Processing Flow}}\label{dir_4b2c7d6f7844b87512b4f74b6a153f60_autotoc_md26}
The audio processing flow contains next blocks\+:
\begin{DoxyEnumerate}
\item Audio processing task
\begin{DoxyItemize}
\item Read left and right channel data from the microphone
\item Apply window multiplication to both channels
\item Process FFT and apply bit reverse
\item Split complex spectrum from two channels to two spectrums of two real channels
\item Calculate absolute spectrum and convert it to dB
\item Calculate moving average of the spectrum
\end{DoxyItemize}
\item Image Display Task
\begin{DoxyItemize}
\item Read data from the
\item Write data from triple buffer to audio codec
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{dir_4b2c7d6f7844b87512b4f74b6a153f60_autotoc_md27}{}\doxysubsection{\texorpdfstring{How to use the example}{How to use the example}}\label{dir_4b2c7d6f7844b87512b4f74b6a153f60_autotoc_md27}
Just flash the application to the ESP32-\/\+S3-\/\+BOX-\/\+Lite development board, and play some music around or start to speak to the board microphones. The display will show the real-\/time spectrum.
