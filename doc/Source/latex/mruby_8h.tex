Get keyword arguments by {\ttfamily \doxylink{mruby_8h_a6659ff35e634812fff48bb6a7f66730e}{mrb\+\_\+get\+\_\+args()}} with {\ttfamily \+:} specifier.

{\ttfamily mrb\+\_\+kwargs\+::num} indicates that the total number of keyword values.

{\ttfamily mrb\+\_\+kwargs\+::required} indicates that the specified number of keywords starting from the beginning of the {\ttfamily \doxylink{classmrb__sym}{mrb\+\_\+sym}} array are required.

{\ttfamily mrb\+\_\+kwargs\+::table} accepts a {\ttfamily \doxylink{classmrb__sym}{mrb\+\_\+sym}} array of C.

{\ttfamily mrb\+\_\+kwargs\+::values} is an object array of C, and the keyword argument corresponding to the {\ttfamily \doxylink{classmrb__sym}{mrb\+\_\+sym}} array is assigned. Note that {\ttfamily undef} is assigned if there is no keyword argument corresponding over {\ttfamily mrb\+\_\+kwargs\+::required} to {\ttfamily mrb\+\_\+kwargs\+::num}.

{\ttfamily mrb\+\_\+kwargs\+::rest} is the remaining keyword argument that can be accepted as {\ttfamily \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}rest} in Ruby. If {\ttfamily NULL} is specified, {\ttfamily Argument\+Error} is raised when there is an undefined keyword.
