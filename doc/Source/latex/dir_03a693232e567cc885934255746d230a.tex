\doxysubsection{Detailed Description}
\hypertarget{dir_03a693232e567cc885934255746d230a_autotoc_md156}{}\doxysubsection{\texorpdfstring{FIR Filter Example}{FIR Filter Example}}\label{dir_03a693232e567cc885934255746d230a_autotoc_md156}
(See the README.\+md file in the upper level \textquotesingle{}examples\textquotesingle{} directory for more information about examples.)

This example demonstrates how to use FIR filter functionality from esp-\/dsp library. Example does the following steps\+:


\begin{DoxyEnumerate}
\item Initialize the FFT library
\item Initialize input signal
\begin{DoxyItemize}
\item 1st Sine wave (f = 0.\+2\+Fs)
\item 2nd Sine wave (f = 0.\+4\+Fs)
\item Combine the waves
\end{DoxyItemize}
\item Show input signal
\begin{DoxyItemize}
\item Calculate windows coefficients
\item Apply the windowing to the input signal
\item Do the FFT
\item Show the frequency response on a plot
\item Calculate execution performance
\end{DoxyItemize}
\item Show filtered signal
\begin{DoxyItemize}
\item Initialize the FIR filter library
\item Calculate Windowed-\/\+Sinc coefficients of FIR filter
\item Apply the FIR filter to the input signal
\item Do the FFT
\item Show the frequency response on a plot
\item Calculate execution performance
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{dir_03a693232e567cc885934255746d230a_autotoc_md157}{}\doxysubsection{\texorpdfstring{How to use the example}{How to use the example}}\label{dir_03a693232e567cc885934255746d230a_autotoc_md157}
\hypertarget{dir_03a693232e567cc885934255746d230a_autotoc_md158}{}\doxysubsubsection{\texorpdfstring{Hardware required}{Hardware required}}\label{dir_03a693232e567cc885934255746d230a_autotoc_md158}
This example does not require any special hardware, and can be run on any common development board.\hypertarget{dir_03a693232e567cc885934255746d230a_autotoc_md159}{}\doxysubsubsection{\texorpdfstring{Configure the project}{Configure the project}}\label{dir_03a693232e567cc885934255746d230a_autotoc_md159}
