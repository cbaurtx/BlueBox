\doxysubsection{Detailed Description}
\hypertarget{dir_52c70609641a06106a381c30d4d0f866_autotoc_md144}{}\doxysubsection{\texorpdfstring{FFT 4 Real Input Example}{FFT 4 Real Input Example}}\label{dir_52c70609641a06106a381c30d4d0f866_autotoc_md144}
(See the README.\+md file in the upper level \textquotesingle{}examples\textquotesingle{} directory for more information about examples.)

This example demonstrates how to use FFT functionality from esp-\/dsp library. Example does the following steps\+:


\begin{DoxyEnumerate}
\item Initialize the library
\item Initialize input signals with 1024 samples\+: one 0 dB, second with -\/20 dB
\item Calculate FFT Radix-\/2 for 1024 complex samples
\item Calculate FFT Radix-\/4 for 1024 complex samples
\item Apply bit reverse operation for output complex vectors
\item Show results on the plots
\item Show execution time of FFTs
\end{DoxyEnumerate}\hypertarget{dir_52c70609641a06106a381c30d4d0f866_autotoc_md145}{}\doxysubsection{\texorpdfstring{How to use example}{How to use example}}\label{dir_52c70609641a06106a381c30d4d0f866_autotoc_md145}
\hypertarget{dir_52c70609641a06106a381c30d4d0f866_autotoc_md146}{}\doxysubsubsection{\texorpdfstring{Hardware required}{Hardware required}}\label{dir_52c70609641a06106a381c30d4d0f866_autotoc_md146}
This example does not require any special hardware, and can be run on any common development board.\hypertarget{dir_52c70609641a06106a381c30d4d0f866_autotoc_md147}{}\doxysubsubsection{\texorpdfstring{Configure the project}{Configure the project}}\label{dir_52c70609641a06106a381c30d4d0f866_autotoc_md147}
Under Component Config -\/-\/-\/\texorpdfstring{$>$}{>} DSP Library -\/-\/-\/\texorpdfstring{$>$}{>} DSP Optimization, it\textquotesingle{}s possible to choose either the optimized or ANSI implementation, to compare them.\hypertarget{dir_52c70609641a06106a381c30d4d0f866_autotoc_md148}{}\doxysubsubsection{\texorpdfstring{Build and flash}{Build and flash}}\label{dir_52c70609641a06106a381c30d4d0f866_autotoc_md148}
Build the project and flash it to the board, then run monitor tool to view serial output (replace PORT with serial port name)\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{idf.py\ -\/p\ PORT\ flash\ monitor}

\end{DoxyCode}


(To exit the serial monitor, type {\ttfamily Ctrl-\/\mbox{]}}.)
