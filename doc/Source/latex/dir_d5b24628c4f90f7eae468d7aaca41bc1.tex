\doxysubsection{Detailed Description}
\hypertarget{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md126}{}\doxysubsection{\texorpdfstring{Basic Math Example}{Basic Math Example}}\label{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md126}
(See the README.\+md file in the upper level \textquotesingle{}examples\textquotesingle{} directory for more information about examples.)

This example demonstrates how to use basic math functions from esp-\/dsp library. Example does the following steps\+:


\begin{DoxyEnumerate}
\item Initialize the library
\item Initialize input signals with 1024 samples
\item Apply window to input signal by standard C loop.
\item Calculate FFT for 1024 complex samples and show the result
\item Show results on the plots
\item Apply window to input signal by basic math functions dsps\+\_\+mul\+\_\+f32 and dsps\+\_\+mulc\+\_\+f32.
\item Calculate FFT for 1024 complex samples
\item Show results on the plots
\end{DoxyEnumerate}\hypertarget{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md127}{}\doxysubsection{\texorpdfstring{How to use example}{How to use example}}\label{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md127}
\hypertarget{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md128}{}\doxysubsubsection{\texorpdfstring{Hardware required}{Hardware required}}\label{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md128}
This example does not require any special hardware, and can be run on any common development board.\hypertarget{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md129}{}\doxysubsubsection{\texorpdfstring{Configure the project}{Configure the project}}\label{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md129}
Under Component Config -\/-\/-\/\texorpdfstring{$>$}{>} DSP Library -\/-\/-\/\texorpdfstring{$>$}{>} DSP Optimization, it\textquotesingle{}s possible to choose either the optimized or ANSI implementation, to compare them.\hypertarget{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md130}{}\doxysubsubsection{\texorpdfstring{Build and flash}{Build and flash}}\label{dir_d5b24628c4f90f7eae468d7aaca41bc1_autotoc_md130}
