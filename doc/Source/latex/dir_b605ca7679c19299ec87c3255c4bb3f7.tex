\doxysubsection{Detailed Description}
\hypertarget{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md150}{}\doxysubsection{\texorpdfstring{FFT Window Example}{FFT Window Example}}\label{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md150}
(See the README.\+md file in the upper level \textquotesingle{}examples\textquotesingle{} directory for more information about examples.)

This example demonstrates how to use Window and FFT functionality from esp-\/dsp library. Example does the following steps\+:


\begin{DoxyEnumerate}
\item Initialize the library
\item Initialize input signals with 1024 samples
\item Apply window to input signal.
\item Calculate FFT for 1024 complex samples
\item Apply bit reverse operation for output complex vector
\item Split one complex FFT output spectrum to two real signal spectrums
\item Show results on the plots
\end{DoxyEnumerate}\hypertarget{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md151}{}\doxysubsection{\texorpdfstring{How to use example}{How to use example}}\label{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md151}
\hypertarget{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md152}{}\doxysubsubsection{\texorpdfstring{Hardware required}{Hardware required}}\label{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md152}
This example does not require any special hardware, and can be run on any common development board.\hypertarget{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md153}{}\doxysubsubsection{\texorpdfstring{Configure the project}{Configure the project}}\label{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md153}
Under Component Config -\/-\/-\/\texorpdfstring{$>$}{>} DSP Library -\/-\/-\/\texorpdfstring{$>$}{>} DSP Optimization, it\textquotesingle{}s possible to choose either the optimized or ANSI implementation, to compare them.\hypertarget{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md154}{}\doxysubsubsection{\texorpdfstring{Build and flash}{Build and flash}}\label{dir_b605ca7679c19299ec87c3255c4bb3f7_autotoc_md154}
Build the project and flash it to the board, then run monitor tool to view serial output (replace PORT with serial port name)\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{idf.py\ -\/p\ PORT\ flash\ monitor}

\end{DoxyCode}


(To exit the serial monitor, type {\ttfamily Ctrl-\/\mbox{]}}.)
