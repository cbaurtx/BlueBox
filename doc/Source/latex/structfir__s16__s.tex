\doxysubsection{Detailed Description}
Data struct of s16 fir filter. 

This structure is used by a filter internally. A user should access this structure only in case of extensions for the DSP Library. All fields of this structure are initialized by the dsps\+\_\+fir\+\_\+init\+\_\+s16(...) function.
