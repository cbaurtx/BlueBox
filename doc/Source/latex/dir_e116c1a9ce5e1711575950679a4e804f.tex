\doxysubsection{Detailed Description}
\hypertarget{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md138}{}\doxysubsection{\texorpdfstring{FFT Example}{FFT Example}}\label{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md138}
(See the README.\+md file in the upper level \textquotesingle{}examples\textquotesingle{} directory for more information about examples.)

This example demonstrates how to use FFT functionality from esp-\/dsp library. Example does the following steps\+:


\begin{DoxyEnumerate}
\item Initialize the library
\item Initialize input signals with 1024 samples\+: one 0 dB, second with -\/20 dB
\item Combine two signals as one complex input signal and apply window to input signals paar.
\item Calculate FFT for 1024 complex samples
\item Apply bit reverse operation for output complex vector
\item Split one complex FFT output spectrum to two real signal spectrums
\item Show results on the plots
\item Show execution time of FFT
\end{DoxyEnumerate}\hypertarget{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md139}{}\doxysubsection{\texorpdfstring{How to use example}{How to use example}}\label{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md139}
\hypertarget{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md140}{}\doxysubsubsection{\texorpdfstring{Hardware required}{Hardware required}}\label{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md140}
This example does not require any special hardware, and can be run on any common development board.\hypertarget{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md141}{}\doxysubsubsection{\texorpdfstring{Configure the project}{Configure the project}}\label{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md141}
Under Component Config -\/-\/-\/\texorpdfstring{$>$}{>} DSP Library -\/-\/-\/\texorpdfstring{$>$}{>} DSP Optimization, it\textquotesingle{}s possible to choose either the optimized or ANSI implementation, to compare them.\hypertarget{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md142}{}\doxysubsubsection{\texorpdfstring{Build and flash}{Build and flash}}\label{dir_e116c1a9ce5e1711575950679a4e804f_autotoc_md142}
