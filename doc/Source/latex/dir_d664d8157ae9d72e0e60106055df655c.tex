\doxysubsection{Detailed Description}
\mbox{[}!\mbox{[}Component Registry\mbox{]}(\href{https://components.espressif.com/components/espressif/esp-dsp/badge.svg}{\texttt{https\+://components.\+espressif.\+com/components/espressif/esp-\/dsp/badge.\+svg}})\mbox{]}(\href{https://components.espressif.com/components/espressif/esp-dsp}{\texttt{https\+://components.\+espressif.\+com/components/espressif/esp-\/dsp}})\hypertarget{dir_d664d8157ae9d72e0e60106055df655c_autotoc_md192}{}\doxysubsection{\texorpdfstring{Espressif DSP Library}{Espressif DSP Library}}\label{dir_d664d8157ae9d72e0e60106055df655c_autotoc_md192}
ESP-\/\+DSP is the official DSP library for \href{https://espressif.com/en/products/hardware/esp32/overview}{\texttt{ESP32}} and \href{https://espressif.com/en/products/hardware/esp32s3/overview}{\texttt{ESP32-\/\+S3}} chips.\hypertarget{dir_d664d8157ae9d72e0e60106055df655c_autotoc_md193}{}\doxysubsubsection{\texorpdfstring{Overview}{Overview}}\label{dir_d664d8157ae9d72e0e60106055df655c_autotoc_md193}
ESP-\/\+DSP is intended to be used as an \href{https://github.com/espressif/esp-idf}{\texttt{ESP-\/\+IDF}} component. For the introduction to ESP-\/\+IDF, refer to the \href{https://docs.espressif.com/projects/esp-idf/en/latest/}{\texttt{ESP-\/\+IDF Programming Guide}}.

The ESP-\/\+DSP library includes implementations of the following functions\+:
\begin{DoxyItemize}
\item Matrix multiplication\+: \href{https://docs.espressif.com/projects/esp-dsp/en/latest/esp-dsp-apis.html\#matrix-operations-apis}{\texttt{reference}}
\item Dot product\+: \href{https://docs.espressif.com/projects/esp-dsp/en/latest/esp-dsp-apis.html\#dot-product}{\texttt{reference}}, \href{https://github.com/espressif/esp-dsp/tree/master/examples/dotprod}{\texttt{example}}
\item FFT\+: \href{https://docs.espressif.com/projects/esp-dsp/en/latest/esp-dsp-apis.html\#fft}{\texttt{reference}}, \href{https://github.com/espressif/esp-dsp/tree/master/examples/fft}{\texttt{example}}
\item IIR\+: \href{https://docs.espressif.com/projects/esp-dsp/en/latest/esp-dsp-apis.html\#iir}{\texttt{reference}}, \href{https://github.com/espressif/esp-dsp/tree/master/examples/iir}{\texttt{example}}
\item FIR\+: \href{https://docs.espressif.com/projects/esp-dsp/en/latest/esp-dsp-apis.html\#fir}{\texttt{reference}}
\item Vector math operations\+: \href{https://docs.espressif.com/projects/esp-dsp/en/latest/esp-dsp-apis.html\#math}{\texttt{reference}}
\item Kalman filter\+: \href{https://docs.espressif.com/projects/esp-dsp/en/latest/esp-dsp-apis.html\#kalman}{\texttt{reference}}
\end{DoxyItemize}

Many of the library functions are written in assembly and are optimized for the CPU configuration used in the ESP32. In addition to the optimized implementations, reference implementations written in ANSI C are provided.

Function implementations are provided for single precision floating point (32-\/bit float), and 16-\/bit signed integers.\hypertarget{dir_d664d8157ae9d72e0e60106055df655c_autotoc_md194}{}\doxysubsubsection{\texorpdfstring{Documentation}{Documentation}}\label{dir_d664d8157ae9d72e0e60106055df655c_autotoc_md194}
