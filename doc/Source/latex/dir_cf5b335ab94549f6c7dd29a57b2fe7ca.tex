\doxysubsection{Detailed Description}
\hypertarget{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md162}{}\doxysubsection{\texorpdfstring{IIR Filter Example}{IIR Filter Example}}\label{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md162}
(See the README.\+md file in the upper level \textquotesingle{}examples\textquotesingle{} directory for more information about examples.)

This example demonstrates how to use IIR filters functionality from esp-\/dsp library. Example does the following steps\+:


\begin{DoxyEnumerate}
\item Initialize the library
\item Initialize input signal
\item Show LPF filter with Q factor 1
\begin{DoxyItemize}
\item Calculate iir filter coefficients
\item Filter the input test signal (delta function)
\item Shows impulse response on the plot
\item Shows frequency response on the plot
\item Calculate execution performance
\end{DoxyItemize}
\item The same for LPF filter with Q factor 10
\end{DoxyEnumerate}\hypertarget{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md163}{}\doxysubsection{\texorpdfstring{How to use example}{How to use example}}\label{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md163}
\hypertarget{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md164}{}\doxysubsubsection{\texorpdfstring{Hardware required}{Hardware required}}\label{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md164}
This example does not require any special hardware, and can be run on any common development board.\hypertarget{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md165}{}\doxysubsubsection{\texorpdfstring{Configure the project}{Configure the project}}\label{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md165}
Under Component Config -\/-\/-\/\texorpdfstring{$>$}{>} DSP Library -\/-\/-\/\texorpdfstring{$>$}{>} DSP Optimization, it\textquotesingle{}s possible to choose either the optimized or ANSI implementation, to compare them.\hypertarget{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md166}{}\doxysubsubsection{\texorpdfstring{Build and flash}{Build and flash}}\label{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md166}
Build the project and flash it to the board, then run monitor tool to view serial output (replace PORT with serial port name)\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{idf.py\ -\/p\ PORT\ flash\ monitor}

\end{DoxyCode}


(To exit the serial monitor, type {\ttfamily Ctrl-\/\mbox{]}}.)

See the Getting Started Guide for full steps to configure and use ESP-\/\+IDF to build projects.\hypertarget{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md167}{}\doxysubsection{\texorpdfstring{Example output}{Example output}}\label{dir_cf5b335ab94549f6c7dd29a57b2fe7ca_autotoc_md167}
