\doxysubsection{Detailed Description}
Data struct of f32 fir filter. 

This structure is used by a filter internally. A user should access this structure only in case of extensions for the DSP Library. All fields of this structure are initialized by the dsps\+\_\+fir\+\_\+init\+\_\+f32(...) function.
