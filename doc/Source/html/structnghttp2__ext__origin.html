Detailed Description

The payload of ORIGIN frame. ORIGIN frame is a non-critical extension to HTTP/2 and defined by RFC 8336 <https://tools.ietf.org/html/rfc8336>_.

If this frame is received, and nghttp2_option_set_user_recv_extension_type() is not set, and nghttp2_option_set_builtin_recv_extension_type() is set for :enum:nghttp2_frame_type.NGHTTP2_ORIGIN, nghttp2_extension.payload will point to this struct.
