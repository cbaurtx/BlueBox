Detailed Description

The payload of PRIORITY_UPDATE frame. PRIORITY_UPDATE frame is a non-critical extension to HTTP/2. If this frame is received, and nghttp2_option_set_user_recv_extension_type() is not set, and nghttp2_option_set_builtin_recv_extension_type() is set for :enum:nghttp2_frame_type.NGHTTP2_PRIORITY_UPDATE, nghttp2_extension.payload will point to this struct.
