◆ name

uint8_t* nghttp2_nv::name

The |name| byte string. If this struct is presented from library (e.g., :type:nghttp2_on_frame_recv_callback), |name| is guaranteed to be NULL-terminated. For some callbacks (:type:nghttp2_before_frame_send_callback, :type:nghttp2_on_frame_send_callback, and :type:nghttp2_on_frame_not_send_callback), it may not be NULL-terminated if header field is passed from application with the flag :enum:nghttp2_nv_flag.NGHTTP2_NV_FLAG_NO_COPY_NAME). When application is constructing this struct, |name| is not required to be NULL-terminated.
